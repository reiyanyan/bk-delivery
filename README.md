# burgerking-delivery

Live Demo [https://bk-delivery.vercel.app/](https://bk-delivery.vercel.app/)  

## Manual Project Setup


Install node dependencies:

  

```Shell

yarn

```

  

Compiles and hot-reloads for development:

  

```Shell

yarn dev

```

  

Compiles and minifies for production:

  

```Shell

yarn build

```
